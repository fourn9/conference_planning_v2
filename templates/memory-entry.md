# MEMORY.md エントリ テンプレ

`MEMORY.md` に追記するときの形式。frontmatter は付けず、`##` 見出しから書く。

```markdown
## <事実・規約のタイトル>

<本文を 1-3 段落で。declarative で書く>

<必要なら箇条書きや表>

**Source**: <観察のきっかけ、任意>
**First observed**: YYYY-MM-DD（任意）
```

## ✓ 良い例

```markdown
## ハードゲート G1-G7 の適用順序

登壇者候補の検証時、G1 から順に G7 まで評価する。1 つでも不通過なら採用しない。
順序を変えると評価者バイアスが入りやすい。

**Source**: ADR-0003（v1）
```

## ✗ 悪い例

```markdown
## TODO: 来週までに G1 を見直す

<- 一時状態なので書かない
```

```markdown
## 常に G1 から評価せよ

<- 命令形なので書かない。「G1 から順に評価するのが慣行」のように観察形で書く
```
